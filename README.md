Real time markdown preview server

## Requirements

- Python 3.5.1+
- [misaka](http://misaka.61924.nl/)
- [pygments](http://pygments.org/)
- [tornado](http://www.tornadoweb.org/en/stable/)
- Modern browser (IE is **NOT SUPPORTED**)
